Remote, Voice & Gesture Controlled Wheel Chair

The Voice-Controlled Wheelchair project aims to develop an innovative assistive technology solution that provides individuals with physical disabilities greater independence and mobility through intuitive voice commands. The wheelchair is designed to respond to spoken instructions, allowing users to control its movements seamlessly.

Smart Intrusion Monitoring System (IMS)

This project features a Smart Intrusion Monitoring System using an ESP32 microcontroller, PIR motion sensor, buzzer, and LED indicators. When motion is detected, the buzzer sounds, a red LED lights up, and a notification is sent via the Blynk app. A green LED shows the system is armed. With real-time alerts and remote access through Blynk, users can monitor and manage home security from anywhere, ensuring safety and peace of mind.
